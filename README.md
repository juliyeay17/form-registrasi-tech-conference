# 🧩 Form Registrasi Tech Conference

## 📌 Deskripsi
Project ini merupakan implementasi form registrasi event menggunakan React Hook Form dengan validasi lengkap (built-in & custom).

Form ini dibuat sebagai bagian dari Project Challenge Front-End Development.

---

## 🚀 Fitur Utama
- ✅ 8 field input lengkap
- ✅ Validasi bawaan React Hook Form
- ✅ Validasi custom (password)
- ✅ Error message dinamis
- ✅ Uncontrolled form (tanpa useState untuk input)
- ✅ Success message setelah submit
- ✅ Auto hide success message (3 detik)

---

## 🧱 Teknologi yang Digunakan
- React JS
- React Hook Form
- Vite
